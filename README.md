# Myshell Phase 3
##기능

-**백그라운드 작업 실행**: '&'를 사용한 기능
-**작업 제어 명령어**:
  - 'jobs': 실행 중인 작업과 정지된 작업 목록 표시
  - 'bg <job>': 정지된 작업을 백그라운드에서 다시 실행
  - 'fg <job>': 백그라운드 작업을 포그라운드에서 실행
  - 'kill <job>': 작업 종료
**신호 처리**:
- 'SIGINT (Ctrl+C)': 포그라운드 프로세스를 종료
- 'SIGTSTP (Ctrl+Z)': 포그라운드 프로세스를 일시 중지
- 'SIGCONT': 중지된 프로세스를 다시 실행
